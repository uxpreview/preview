Components

Forty-five components, nine categories.

Every component in the system, grouped by what it does. Each one carries an anatomy diagram, a spec sheet, written guidelines, and an accessibility profile. Click a card to start.

Actions

Things the user can click, tap, or trigger.

Stable

Button

A pill-shaped action with three emphasis levels and three sizes.

Forms & inputs

Controls for collecting and editing user input.

Migrating

Field

The label + input + hint + error wrapper. Every form control composes through this.

Migrating

Input

Single-line text input. Pairs with Field for label, hint, and error.

Migrating

Textarea

Multi-line text input for longer freeform responses.

Migrating

Select

A native dropdown for choosing one option from a known list.

Migrating

Checkbox

A single binary choice or one option in a multi-select group.

Migrating

Radio

A mutually-exclusive choice within a small set of options.

Migrating

Toggle

An instant on/off switch for boolean settings.

Selection & status

Status labels and interactive selection chips.

Stable

Badge

A compact status indicator, with variants, a dot, and a count overlay.
Stable

Tag

An interactive filter / removable chip and a read-only metadata label, in two sizes.

Containment & overlays

Surfaces that group, disclose, or layer content.

Migrating

Card

A self-contained surface for an article, person, resource, or stat.

Stable

Accordion

Disclosure groups using native <details> for FAQs and progressive content.
Migrating

Tabs

The ARIA APG tab pattern with hash deep-linking and pill variant.

Migrating

Modal

A centered dialog with focus trap and ARIA dialog semantics.

Migrating

Drawer

An off-canvas panel with focus trap, used on mobile and as a side sheet.

Stable

Divider

A horizontal or vertical rule for separating content, with weight and style variants.

Navigation

Wayfinding between and within pages.

Migrating

Breadcrumb

A trail showing the user's position in a hierarchical structure.

Migrating

Pagination

Numbered page navigation for long lists and result sets.

Migrating

Topnav

The sticky primary navigation at the top of every page.

Migrating

Megamenu

A wide dropdown panel for sites with deep section hierarchies.

Migrating

Sidenav

A vertical navigation for local sections, with sublist support.

Migrating

In-page nav

A scroll-spy nav that highlights the current section on long pages.

Layout

Page-level structure and section scaffolding.

Migrating

Two-column

A balanced two-column layout for headline + supporting content.

Migrating

Feature grid

A row of equally-weighted features or capabilities.

Migrating

Hero

The opening block at the top of a page, in five variants.

Migrating

Footer

Four variants: compact, standard, expanded, inverse.

Content & display

Presenting text, media, lists, and tables.

Migrating

Callout

A boxed aside for asides, notes, and inline emphasis.

Migrating

Quote

A pull quote with optional attribution and emphasis treatment.

Migrating

Timeline

A vertical sequence of dated milestones or steps.

Migrating

Media

A placeholder for imagery with explicit aspect ratio labels.

Migrating

Gallery

Grid and mosaic layouts for collections of media.

Migrating

Linked list

A list of links with hover affordance, for nav-adjacent lists.

Migrating

Definition list

Term + definition pairs, semantic and screen-reader friendly.

Migrating

Article list

A list of articles or posts with title, summary, and metadata.

Migrating

Citation list

Numbered citations for research-backed long-form pages.

Migrating

Table

A striped data table with proper heading semantics.

Messaging & feedback

Communicating status, progress, and system messages.

Migrating

Toast

Transient status messages via a polite live region.

Migrating

Banner

A persistent inline message for warnings, info, or success.

Migrating

Empty state

A blank-slate prompt when a list or page has no content yet.

Migrating

Stepper

A numbered progress indicator for multi-step flows.

Migrating

Tooltip

A small contextual hint attached to a control.

Migrating

Skeleton

Loading-state placeholders that match the layout that's coming.

Patient affordances

Healthcare aids for older and access-sensitive audiences.

Migrating

Help bar

A persistent contact bar for time-sensitive support paths.

Migrating

Phone link

A tap-to-call number formatted for clarity and accessibility.

Migrating

Text size

A user-controlled text scale with persistence and SR announcement.
