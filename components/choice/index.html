Choice

Stable

Controls for picking from options: a .wire-checkbox for any-number, a .wire-radio for exactly-one, and a .wire-toggle for an instant on/off setting. Each is a label wrapping a styled native input, so the whole row is clickable.

  • css/components/form.css
  • Since v1.0

Live demo

Each control is a <label> wrapping its input, so clicking the text toggles it. Related choices group inside a <fieldset> with a legend.

Which one

The shape of the answer picks the control.

ControlAnswer shapeUse for
Checkboxzero or moreMulti-select filters, or a single yes/no agreement.
Radioexactly oneOne choice from a small, fully-visible set (2–5).
Toggleon / offAn immediate setting that takes effect with no save step.

Use radios, not a select, when the set is small and seeing every option matters. Use a toggle only when the change is instant — if it needs a Save or confirmation, use a checkbox.

In context

Choices cluster in filter panels and settings. Group with a fieldset; keep the legend short.

A filter and a setting
Visit type

Do / Don't

Do

  • Wrap each control in a <label> so the text is clickable.
  • Group related radios/checkboxes in a <fieldset> with a legend.
  • Give every radio in a set the same name.

Don't

  • Use a toggle for a change that needs a Save or confirmation.
  • Use radios for a long list — switch to a select past ~5 options.
  • Use a lone checkbox where exactly one of several must be chosen.

Grouping

  • Wrap a set of related radios or checkboxes in a <fieldset> with a <legend> — the legend names the group for assistive tech.
  • Every radio in one group shares the same name, which makes them mutually exclusive and arrow-navigable.

Labels & target

  • Each control is a <label> wrapping its input, so the text is part of the control — no separate for/id needed, and the whole row is clickable.
  • The row keeps a 44px-tall target even though the box itself is smaller.
  • Checked state is carried by the control's shape and fill, never color alone (WCAG 1.4.1).

Keyboard

KeyAction
SpaceToggles a checkbox or toggle; selects the focused radio.
ArrowsMove and select within a radio group.
TabEnters the group at the checked radio, or each checkbox in turn.

All controls keep the visible focus ring; disabled controls are skipped in the tab order.
