Phone link

Stable

A tap-to-call phone number — a tel: link with a leading icon, formatted for clarity and sized to tap. It turns a displayed number into a one-tap call on mobile and a one-click handoff to the dialer on desktop.

  • css/components/help-bar.css
  • 1 part + icon

Live demo

A phone number rendered as a tappable link: a leading call icon, then the human-formatted number. The whole pill is the tap target.

Hospital main line ☏ (312) 555-0142

Anatomy

One link wrapper holds a decorative icon and the visible number. There are no variants — the part stays constant wherever a number appears.

PartClassRole
Link wrapper.wire-phone-linkThe <a href="tel:…">; carries the tap target and is the accessible name.
Icon.wire-phone-link__iconLeading call glyph; decorative, marked aria-hidden="true".

When to use

  • Use it for any displayed phone number a visitor might call — a main line, a department, a 24/7 nurse line, an admissions desk.
  • Pair it with surrounding context that says who and when — a contact section, a profile, or a Help bar that frames the call before the visitor makes it.
  • Reach for it instead of plain text whenever the number is meant to be dialed; on mobile it saves the visitor from copying digits by hand.

For a louder, full-width call-to-action, a standard Button with a tel: href reads as the primary action; the phone link is the quieter, inline treatment.

In context

The same part works inline inside a sentence and on its own as a standalone affordance.

Inline in a sentence

Patient navigators are available 8 AM – 6 PM weekdays. Call ☏(312) 555-0148 — there's no charge to talk.

Standalone — 24/7 nurse line ☏ (800) 555-0199

Do / Don't

Do

  • Format the number for reading — group the digits the way a person would say them: (312) 555-0142.
  • Keep the human number as the visible text, so it's the accessible name a screen reader announces.
  • Place it near context that names the line and its hours.

Don't

  • Hide the number behind an icon-only control — the digits must stay visible and readable.
  • Put the number only in the visible text without a tel: href — it won't dial on tap.
  • Cram the digits together with no grouping; an unbroken string is harder to read and to trust.

Formatting

The href and the visible text carry the same number in two forms: machine-dialable digits in the href, human-readable grouping in the label.

SlotFormExample
hrefE.164 digits with country code, no spaces or punctuation.tel:+13125550142
Visible textHuman-formatted, grouped the way it's spoken.(312) 555-0142

North American grouping follows Formatting. Digits render in tabular figures so numbers line up wherever they repeat.

Tokens

PropertyToken
Min tap heightmin-block-size: 2.75rem (44px)
Padding--space-xs --space-md
Icon–text gap--space-2xs
Label type--text-md, --weight-semibold, tabular numerals
Icon size--text-lg
Text color--color-text
Shape--radius-pill
Hover surface--color-bg-muted

Snippet

<a class="wire-phone-link" href="tel:+13125550142">
  <span class="wire-phone-link__icon" aria-hidden="true">☏</span>
  (312) 555-0142
</a>

Target size

  • The tappable link meets WCAG 2.2 SC 2.5.8 Target Size (Minimum), which requires a 24×24 CSS-pixel target (standard).
  • It also clears the system's ≥44px convention: the link sets min-block-size: 2.75rem (44px) so it's comfortable for older hands and touch input.
  • Padding extends the hit area to the full pill, not just the text — the icon and the whitespace around the digits are tappable too.

Labeling

  • The visible number is the accessible name — a screen reader announces the digits a user would dial, with no separate aria-label needed.
  • The leading icon is decorative and marked aria-hidden="true", so it isn't read out before the number.
  • Keep the human-formatted number in the text; don't replace it with a word like "Call us" that hides which number is dialed.

Semantics

  • It's a genuine tel: link, so mobile and desktop dialers and assistive tech recognise it as a phone number and offer to call (the tel: URI scheme, RFC 3966).
  • Being a real <a>, it's keyboard-focusable, shows the system focus ring, and works with standard link navigation — no custom JavaScript.
  • The E.164 href carries the country code, so the number dials correctly regardless of where the caller is.
