Link

Beta

A text hyperlink: an <a href> that navigates to another page, resource, or location. Underlined and inheriting the text color, it is the system's quietest action. Use a link to go somewhere; use a Button to do something.

  • css/base.css (bare a)
  • Since v1.0

Live demo

A link is a plain <a href>. It is underlined and uses the body text color, so it reads as a link without relying on hue.

Inline and standalone

Stroke rehabilitation begins with an assessment from our inpatient therapy team, usually within 24 hours of admission.

View all rehabilitation services

Anatomy

A link has no component wrapper or class — it is the native anchor styled in the base layer. Its meaning lives in the href and the text between the tags.

PartMarkupRole
Anchor<a href="…">The native link element; the href is the destination.
Link texttext contentThe accessible name and the visible label — it must describe the destination.
Underlinebase styleThe non-color signal that the text is a link (added in css/base.css).

When to use

  • Use a link to navigate — to another page, a document, an anchor on the page, or an external site.
  • Use a Button to perform an action — submit, open a dialog, save, delete. The element should match the behavior, so keyboard and screen-reader users get the right semantics.
  • Use a Phone link for a tap-to-call number, and a Linked list when links form a scannable, large-target set of rows.

If a control looks like a link but acts like a button (or the reverse), the underlying element still has to match what it does.

In context

The same anchor serves inline prose, standalone calls to action, list rows, and external destinations.

Inline within body text

Most plans cover therapy after a qualifying stay. Check your coverage and referral requirements before scheduling.

As a row in a linked list
  • Spinal cord injury14 conditions
  • Stroke22 conditions
External link, with an in-text cue

Eligibility follows the state program: read the benefits handbook ↗ (opens in a new tab).

Tap-to-call number (Phone link)

Questions about a bill? Call ☏(312) 555-0142.

Do / Don't

Do

  • Write link text that describes its destination and front-loads the meaningful words.
  • Keep underlines on links inside body text so they're distinguishable without color.
  • Say when a link opens a new tab or a file, in the link text itself.

Don't

  • Use "click here", "read more", or a bare URL as the link text.
  • Underline or color non-link text so it looks clickable.
  • Open new tabs by default, or trigger an action from a link.

States

A link carries state through weight and a focus ring, never hue. Hover and focus are shown here for reference; move through the demo with the keyboard to see the focus treatment.

Rest

An appointment with the neurology intake team can be booked online.

StateTreatment
RestBody text color, 1px underline offset from the baseline.
HoverUnderline thickens to 2px — a weight change, not a color change.
Focus-visibleThe global focus ring; visible on keyboard focus.
VisitedNot restyled by default; the system relies on text and context, not a visited hue.

Snippet

<!-- Inline, descriptive link text -->
<a href="/services/rehabilitation">rehabilitation services</a>

<!-- External link: signal the new tab in the text -->
<a href="https://example.gov/benefits" target="_blank" rel="noopener">
  benefits handbook <span aria-hidden="true">↗</span> (opens in a new tab)
</a>

Labeling

  • The link's purpose should be clear from its text alone, so it makes sense when read out of context — a screen reader can list every link on the page with no surrounding sentence.
  • Keep link text short and specific. Avoid "click here", "read more", and bare URLs, which give no destination.
  • If a link is unavoidably ambiguous, add a clarifying aria-label or visually-hidden text rather than leaving the bare phrase.

Semantics

  • Use <a href> for navigation and <button> for in-page actions; don't fake one with the other (href="#" handlers report the wrong role to assistive tech).
  • A link must be distinguishable from surrounding text by more than color — the underline carries that signal, which is essential in this grayscale system.
  • If a link opens a new tab or window, signal it in the link text; an unexpected context change is disorienting, especially for screen-magnifier users.

Keyboard

KeyAction
TabMoves focus to and from the link in source order.
EnterActivates the link and navigates to its destination.

Focus stays visible on keyboard navigation. A standalone link keeps a target of at least 44px; inline links within a sentence are exempt from the minimum target size.
