Hero

Stable

The first block on a page — an eyebrow, title, optional lead, and an actions row. Five variants cover the common openings: editorial for big-type statements, split for image-and-text, centered for landing pages, stat-led when numbers carry the message, and inverse to flip any of them onto a dark surface.

  • 5 variants

Anatomy

A hero stacks an eyebrow, title, optional lead, and an actions row inside a centered inner column. Variants change how those parts lay out; the parts themselves stay constant. Stat-led adds a stats row; split adds a media slot.

PartClassRole
Container.wire-hero + variantThe block; takes one variant modifier.
Inner.wire-hero__innerCentered, width-capped content column.
Eyebrow.wire-hero__eyebrowShort kicker above the title.
Title.wire-hero__titleThe headline (the page h1 in real use).
Lead.wire-hero__leadOptional supporting sentence.
Actions.wire-hero__actionsRow of buttons (primary + secondary).
Media.wire-hero__mediaImage slot — split variant only.
Stats.wire-hero__stats, __stat-value, __stat-labelThree-up figures — stat-led variant only.

When to use

  • Use editorial for a single emphatic landing statement; italics inside the title turn serif for an editorial moment.
  • Use split when an image is part of the message — a patient photo, lab shot, or product image.
  • Use centered for short-form landing pages, campaigns, or single-purpose marketing pages.
  • Use stat-led when three numbers carry more weight than a paragraph — an annual report or research overview.
  • Use inverse sparingly, once per page at most, for a high-emphasis opening or section anchor.

For a lighter page-top label on detail pages, reach for a Page header instead — the hero is for the first, most emphatic block.

Do / Don't

Do

  • Use the hero as the first block, and make its title the page h1.
  • Keep the actions row to one primary action plus an optional secondary.
  • Reserve inverse for a single high-emphasis moment per page.

Don't

  • Stack heroes or use one as a recurring section header.
  • Put meaning in the editorial italic alone — it's decoration.
  • Crop split media to portrait; the layout assumes 16:9 / 4:3 / square.

Headings

  • The hero title is the page's h1 — one per page. Don't place a hero below another h1 as a section header.
  • Keep the eyebrow as text, not a heading; it labels the title, it isn't a level in the outline.

Semantics

  • Split places the media after the text in the DOM, so screen readers hear the message before the image.
  • The media slot is decorative scaffolding (aria-hidden="true"); a real image needs meaningful alt text, or empty alt if purely decorative.
  • Editorial italic is typographic emphasis only — never the sole carrier of meaning.

Contrast

  • The inverse variant keeps body and lead text at or above 4.5:1 on its dark surface (WCAG 2.2 SC 1.4.3).
  • Secondary buttons restyle for the inverse surface so their border and label stay legible.
  • Hierarchy comes from scale, weight, and space — never color — so the block reads the same in light and dark themes.
