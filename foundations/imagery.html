Imagery

This is a grayscale system, so "imagery" means placeholders, not photographs. The page's job is to reserve the right space and describe what the real image will be — its ratio, its content, and its alt text — so production has everything it needs.

Placeholders, not photos

The Media component renders a greybox at a fixed ratio. Its data-ratio prints a corner label, and an optional .wire-media__label names the intended image — a brief for whoever sources the real asset.

Default (16:9) with a label
Hero image — patient on therapy treadmill

Aspect ratios

Pick a ratio for a slot and hold it across the set — every provider portrait square, every story card 16:9 — so the grid stays even.

RatioClassTypical use
16:9wire-media--16-9Heroes, video, story cards.
4:3wire-media--4-3Editorial and content blocks.
1:1wire-media--squareAvatars, thumbnails, logos.
Portraitwire-media--portraitProvider portraits, posters.
Widewire-media--wideBanners and page headers.

Alt text

Every image is either informative or decorative, and the alt attribute follows from that. (WCAG 2.2 — 1.1.1 Non-text Content — a requirement.)

Informative
Concise alt describing the meaning or function, not the pixels. Skip "image of".
Decorative
Empty alt="" so assistive tech skips it; greyboxes carry aria-hidden.
In wireframes
The wire-media__label is the production brief; the real alt is authored when the asset is placed.

Figures and galleries

When an image needs a visible caption — a chart, a diagram, a credited photo — wrap it in a semantic <figure> with a <figcaption>. The caption is shown context; the alt is still required and does a different job.

For several placeholders together, wire-gallery (with --2 / --3 / --4 / --mosaic) lays them out in a grid:

wire-gallery--3, square

Do and don't

Do
Name the intended image in the label, so the greybox is a real brief.
Do
Hold one ratio across a set of related placeholders.
Don't
Bake text into an image — it can't translate or be read aloud. See Localization.
Don't
Write "image of…" in alt, or leave an informative image with empty alt.
