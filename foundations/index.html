Tier 1 · Foundations

The rules the whole system rests on.

Before any component: the principles that govern decisions, the tokens that hold every value, the layout primitives, and the evidence base that makes the system defensible in a client room.

Principle

Principles

The ten governance rules every change follows — curate over add, content not style, honesty over polish.
Tokens

Tokens

The only file with raw values: grayscale ramp, type and spacing scales, radii, borders, shadows, motion. Everything references these.
Layout

Layout & utilities

Container, stack, cluster, grid, sidebar, and the u-* helpers that compose every page without bespoke CSS.
Evidence

Accessibility & evidence

The WCAG 2.2 AA compliance map and the four-tier evidence base — standard, empirical, convention, judgment — behind each decision.
Quality

Validation

The checklist run on every generated page — grayscale, tokens-only, no invented classes, a11y holds — before it ships.
Lifecycle

Component status

What's safe to build on and what's still settling — stable, beta, alpha, deprecated — across every component.
