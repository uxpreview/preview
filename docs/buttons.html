Component

Buttons

A pill-shaped action with three emphasis levels and three sizes. Use exactly one primary per view and reach for ghost in dense interfaces.

Anatomy

A button is a single rounded element with an optional leading or trailing icon, a label, and a minimum 44px touch target on every size.

Default

When to use

  • Use primary for the single most important action in a view (one per region).
  • Use secondary for the supporting action paired with a primary.
  • Use ghost in toolbars, card footers, and dense surfaces where a bordered button would feel heavy.
  • Use icon-only for utility actions where the icon's meaning is unambiguous and an aria-label is provided.

When not to use

  • Do not use a button when navigating; use an <a> styled as a button instead.
  • Do not stack three or more primary buttons; rework the page hierarchy.
  • Do not use icon-only buttons for destructive actions without confirmation.

Variants

Three emphasis levels, three sizes, plus modifiers for icons, full width, and loading state.

Emphasis — primary, secondary, ghost
Sizes — sm, md (default), lg
With icon — leading and trailing
Icon-only — sm, md, lg
Full width
Loading
Disabled
Button group

Accessibility

  • Always use <button type="button"> outside forms and type="submit" for primary form actions.
  • Icon-only buttons require an aria-label; the icon element should be aria-hidden="true".
  • Every size meets a 44×44 touch target except --sm, which is intended for dense toolbars and meets 36×36 with adequate spacing.
  • Focus is rendered as a 2px outline with offset; never remove it.
  • Loading state sets pointer-events: none and hides label text; pair with aria-busy="true" when wiring up.

Code

Show snippet 
<button class="p-button" type="button">Make an appointment</button>
<button class="p-button p-button--secondary" type="button">Learn more</button>
<button class="p-button p-button--ghost p-button--sm" type="button">Cancel</button>
<button class="p-button p-button--icon-only" type="button" aria-label="Search">
  <span aria-hidden="true">⌕</span>
</button>
