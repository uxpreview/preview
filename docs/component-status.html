v1.8 · Component status

What's safe to build on, what's still settling.

Every component carries a status. Stable means the API is frozen until the next major version. Beta means it works and is used in real demos, but the class names or markup shape may refine. Alpha means experimental — don't use in client work. Deprecated means scheduled for removal at v2.0.

Status legend

Stable API frozen until v2.0 Beta Working, in demos, API may refine Alpha Experimental Deprecated Scheduled for v2.0 removal

Promotion rules in CONTRIBUTING.md: alpha → beta after two demo-page uses without API change; beta → stable after a documented WCAG audit pass.
