Tier 5 · Experiences

Directories and flows — the tier that makes generation intentional.

An experience is a template plus a content model plus an interaction pattern, captured as named variants and a recipe. It's where a request resolves to a real IA decision instead of a plausible pile of blocks.

Status

Not built yet — this tier is the roadmap.

The pieces below the line — components, patterns, templates — exist today. The Experiences tier composes them and is built last (see the brief, §10 Roadmap). The two planned first are scoped here so the tier is ready to fill.

Planned experiences

Planned

Find-a-doctor directory

Help a visitor locate the right clinician and act. Built on search, tag filters, person cards, and pagination.

  • filter-left — large directory, many facets
  • search-first — arrives with a name in mind
  • browse-by-specialty — leads with programs

Closest demo today: hospital-find-a-doctor.

Plan-your-visit

Reduce anxiety and answer logistics before a first appointment. Built on hero, accordion, definition list, and the help bar.

  • faq-led — questions dominate
  • step-by-step — referral → schedule → arrive

Closest demo today: hospital-patient-visitor.
